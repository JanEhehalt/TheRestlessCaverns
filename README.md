## Controls

**W**: move up  
**A**: move left  
**S**: move down  
**D**: move right  
**SPACE**: Attack  
**F**: pick up  
**R**: equip selected item  
**scroll-wheel**: change selected item  
**Q**: drop selected Item  
**ESC**: pause Game / resume Game
